  1. nexus-2.11.4-01-bundle版本
  2. 上传并且解压 ，chmod 777 -R nexus-2.11.4-01 chmod 777 -R sonatype-work 授权这两个文件夹权限
  3. 执行 nexus-2.11.4-01\bin 下的nexus 执行文件 ./nexus start 默认启动8081
  4. 此时会出现错误信息 If you insist running as root, then set the environment variable RUN_AS_USER=root…
  5. 添加 RUN_AS_USER=root 环境变量 vi /etc/profile 向其中加入exportRUN_AS_USER=root
  6. 开放防火墙的8081端口  
firewall-cmd --zone=public --add-port=8081/tcp --permanent #开放8081端口  
systemctl restart firewalld.service #重启防火墙

  7. 访问 http:// IP : 8081/nexus 即可
