报错信息如下：

    
    
    status: Unable to connect to system bus: Failed to connect to socket /var/run/dbus/system_bus_socket: No such file or directory

解决方案：

    
    
    yum install -y dbus  #CentOS
    apt-get install dbus  #Debian
